## Overview
This repository contains the code for the Shopify theme used by the Hiatus Craft House distillery for their website and digital storefront. The theme being used is the free theme [Crave](https://themes.shopify.com/themes/crave/presets/crave) created by Shopify, modified for the distillery's use. 

## About [Hiatus Craft House](https://hiatuscrafthouse.com/)
Hiatus Craft House is a craft distillery located in Camarillo, CA. They distill and sell a variety of different types of spirits, with dedication to non-profit work and philanthropy. For more information, see [here](https://hiatuscrafthouse.com/pages/our-story).
